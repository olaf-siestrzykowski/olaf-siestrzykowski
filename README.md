# Hi, I'm Olaf 👋

I started programming in June 2022, coming from a civil engineering background, and freelanced on web projects while building up my skills. From there I moved into an Analyst role, using Python and reporting tools to work with sales data and dashboards, and then into a Python Developer role where I started building automation and scheduled jobs.

I specialise in script automation, ETL pipelines, and business reporting. I build automated data flows that pull from APIs, internal business systems, databases, and web sources - clean and transform the data, then deliver it as Power BI dashboards and structured reports for decision-makers.

📍 Warsaw, Poland · [LinkedIn](https://linkedin.com/in/olaf-siestrzykowski) · [Portfolio](https://olafsiestrzykowski.eu.pythonanywhere.com) · ✉ olaf.siestrzykowski@gmail.com

## What I do

- 🖥️ **Full-stack apps** - Django apps with real-time features, complex data modeling, and automated test suites, built and deployed end-to-end (Docker, Nginx, SSL)
- 🌐 **Backend** - Django / Flask / FastAPI apps, deployed with Docker + Nginx
- 🔁 **Automation** - replace manual, repetitive data work with reliable scheduled scripts (LaunchDaemon, cron, APScheduler, Railway) and API integrations
- 📊 **Reporting** - Power BI, Looker Studio, and Excel (Power Query, VBA) dashboards that decision-makers actually use
- 🧹 **ETL pipelines** - extraction (REST APIs, Playwright, BeautifulSoup) -> cleaning & transformation (Pandas) -> delivery

## 🔭 Currently building

- A full-stack Django app with real-time features for the events space - works both as a one-off tool for running a single event and as a day-to-day companion throughout the preparation period. Launch coming soon.
